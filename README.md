# 🎓 NGBHS Batch ’22 — Student Platform

A full-stack web platform built for **NGBHS Batch 2022**, designed to centralize communication, events, discussions, and donation management in one place.

---

## 🚀 Status

**Currently in development** — following a structured phase-based roadmap.

---

## 🧠 Overview

This project aims to create a **complete digital ecosystem for students**, including:

* 📢 Notice board for announcements
* 💬 Discussion & communication system
* 📩 Messaging and group chat
* 💸 Donation & fundraising system
* 👑 Full admin control panel

The system is designed to be **scalable, modular, and easy to maintain**.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (React)
* **Styling:** Tailwind CSS
* **Backend:** Node.js (optional API layer)
* **Database:** Firebase (Firestore + Realtime DB)
* **Authentication:** Firebase Auth
* **Deployment:** Vercel
* **Payments:** bKash & Nagad

---

## ✨ Core Features

### 🔐 Authentication

* Login using **Roll Number + Section**
* Role-based access (Admin / Student)

### 📋 Notice Board

* Events, announcements, urgent notices
* Category tagging system
* Admin can post, edit, delete, and pin notices

### 💬 Discussion System

* Public and section-based threads
* Reactions (👍 / 🔥)
* Admin moderation tools

### 📩 Chat System

* Direct messaging (DM)
* Batch-wide group chat
* Online status system

### 💰 Donation System

* Fundraising campaigns
* bKash & Nagad payment support
* Transaction submission & verification
* Donor tracking (optional leaderboard)

### 👑 Admin Panel

* Manage students, events, and notices
* Assign roles (Super Admin / Co-admin / Moderator)
* Control payment info and system settings

```bash

# Navigate into project
```
